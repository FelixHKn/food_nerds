# food_nerds

A shared space for our group to compare and rank food products across categories.

## Rankings

- [Vegan bolognese sauces](rankings/vegan-bolognese.md)

## How it works

Each category lives in its own file under `rankings/`. Every product has:

- **Facts** from the label (sugar, protein, additives, cost per 100 g) — filled in once.
- **Tasting scores** — one column per group member for the subjective criteria (e.g. flavor, 1–10).
- A **leaderboard** at the bottom of the file summarizing the group's current view.

To add a new category, copy an existing file in `rankings/` and adapt the criteria.
